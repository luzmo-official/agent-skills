
## Canonical Sources

- Index: `https://developer.luzmo.com/llms.txt`, `https://developer.luzmo.com/llms-full.txt`
- API specs (markdown): `https://developer.luzmo.com/api/{action}{Resource}.md`
- API call forms (HTML, per language): `https://developer.luzmo.com/api/{action}{Resource}/call/{js|python|java|dotnet|curl|php}`
- API examples (HTML, walkthroughs): `https://developer.luzmo.com/api/{action}{Resource}/examples/{slug}/{lang}`
- Guides: `https://developer.luzmo.com/guide/*.md`
- MCP: `https://developer.luzmo.com/guide/mcp--introduction.md`
- Flex charts: `https://developer.luzmo.com/flex/charts/{type}.md`

**Filename casing:** AI resources use CamelCase in `.md` URLs (e.g. `createAIPrompt.md`, not `createAiprompt.md`).

If content exists on developer.luzmo.com, link — do not duplicate specs here.
